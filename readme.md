# Proyecto: Práctica de Listas en HTML

## Descripción

Este proyecto consiste en un ejercicio práctico enfocado en el aprendizaje y uso de **listas en HTML**. A través de dos páginas web, se muestran distintos tipos de listas y su aplicación en contenido estructurado, utilizando además estilos CSS para mejorar la presentación visual.

El objetivo principal es comprender cómo funcionan:

- Listas desordenadas (`<ul>`)
- Listas ordenadas (`<ol>`)
- Listas de definición (`<dl>`)

---

## Objetivos del Proyecto

- Aprender la sintaxis básica de las listas en HTML.
- Diferenciar entre los distintos tipos de listas.
- Aplicar estilos CSS a listas.
- Organizar contenido de forma estructurada y semántica.
- Utilizar enlaces internos dentro de listas.
- Combinar listas con otros elementos HTML (imágenes, texto, etc.).

---

## Contenido del Proyecto

El proyecto incluye **dos páginas HTML**:

### Página principal: "Los Colores"

Una página más completa que utiliza listas para organizar información sobre colores.

#### Características:

- Uso de listas desordenadas para navegación interna.
- Uso de listas ordenadas para clasificar colores.
- Uso de listas de definición para explicar términos.
- Enlaces internos (`anchor links`) entre secciones.
- Inclusión de imágenes relacionadas con cada color.
- Estilos CSS personalizados:
  - Colores de fondo
  - Tipografía externa (Google Fonts)
  - Diseño responsive básico
  - Clases para representar colores visualmente

#### Ejemplo de lo que se trabaja:

- Agrupación de colores por categorías (rojo, verde, azul, etc.)
- Explicación de matices
- Uso de clases CSS para dar estilo a cada elemento de lista

![Demo Listas](/media/demo-listas.gif)

---

### Página secundaria: "Listas"

Una página sencilla orientada a demostrar los diferentes tipos de listas en HTML.

#### Características:

- Listas desordenadas básicas.
- Listas ordenadas con distintos formatos:
  - Numéricas
  - Letras (mayúsculas y minúsculas)
  - Números romanos
- Listas de definición con descripciones de colores.
- Uso del atributo `type` en listas.
- Uso del atributo `start` en listas ordenadas.

## Conceptos Clave Aprendidos

- `<ul>` → Lista desordenada
- `<ol>` → Lista ordenada
- `<dl>` → Lista de definición
- `<li>` → Elemento de lista
- `<dt>` → Término
- `<dd>` → Definición
- Atributos:
  - `type`
  - `start`
- Uso de clases en CSS para estilizar listas
